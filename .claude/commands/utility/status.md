# /status

프로젝트의 전체 진행 상황을 대시보드 형태로 보여줍니다.

## 개요

한눈에 파악할 수 있는 정보:
- 스펙 완성도
- 구현 진행률
- 다음 할 일
- 주의가 필요한 항목

## 프로세스

````
│                                     │
│ 3. /implement TASK-011              │

````

## 옵션
````
/status              # 전체 현황
/status --specs      # 스펙 현황만
/status --tasks      # 태스크 현황만
/status --quality    # 품질 현황만
````

## 파일이 없는 경우
````
📊 프로젝트 현황

⚠️ 스펙 파일이 없습니다.

시작하려면:
1. /spec-discovery 로 Discovery 단계 시작
2. 또는 기존 프로젝트 폴더인지 확인

현재 디렉토리: /Users/jazz/projects/paper-reader
````
